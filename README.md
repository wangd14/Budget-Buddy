## Building
* Make sure node is at version ``v20.11.0``
* log into vm
* ``cd /var/www/html/AbsolutelyNot/backend``
* ``sudo env "PATH=$PATH" npm i``
* ``sudo env "PATH=$PATH" node server.js``
